# GURPS 4e — Session Procedures

## Combat Resolution Tiers

| Tier | Name | When to Use |
|------|------|-------------|
| Quick | Single roll | Mook encounters |
| Basic | Turn-by-turn, no map | Most fights |
| Tactical | Full rules, map | Boss fights, set pieces |

## Basic Combat Maneuvers

| Maneuver | Effect | Defenses |
|----------|--------|----------|
| Attack | Roll vs skill to hit | Dodge, Parry, Block |
| All-Out Attack | +4 to hit, or two attacks | None this turn |
| All-Out Defense | +2 to one defense | Full |
| Aim | +Accuracy on next ranged attack | Full, lost if damaged |
| Evaluate | +1 to next melee attack (max +3) | Full |
| Feint | Quick Contest; penalises defence | Full |
| Move and Attack | Full move, attack at -4 | Dodge only |
| Wait | Interrupt on trigger | Full |

**Simplification strategies:**

- Skip hit locations unless dramatically important.
- Apply -1 per complicating factor instead of table lookups.
- Enforce 5-second decision timer.
- Use BAD (Basic Abstract Difficulty) for non-combat modifiers.

## Skill Challenge Design

**Quick Contest** (instantaneous opposed actions): Both roll
3d6 vs skill. Higher margin of success wins.

**Regular Contest** (prolonged): Both roll each round. Victory
only when one succeeds AND the other fails.

**Complementary skills:** Assistant rolls related skill. Success
gives +1 to primary roll. Critical success gives +2.

## Character Advancement

| Campaign Style | CP/Session | Effect |
|---------------|-----------|--------|
| Gritty | 1-2 | Slow growth |
| Standard | 3-5 | Steady improvement |
| Cinematic | 5-8 | Rapid growth |

## Typical GURPS Session Flow (by Genre)

**Action:**
Briefing -> Planning -> Infiltration -> Tactical encounter ->
Extraction.

**Investigation:**
Hook -> Research -> Fieldwork -> Confrontation -> Resolution.

**Social/Political:**
Situation briefing -> Networking -> The event -> Complications ->
Aftermath.

## Arc Drivers

Arc fuel: disadvantages are built-in story hooks — Duty/Sense
of Duty (obligation conflicts), Dependents (threatened NPCs
per frequency roll), Enemy (recurring antagonist), Obsession/
Compulsive/Addiction (escalating internal conflict), Secret
(ticking time bomb), Vow/Code of Honour (moral constraints).
Rotate which disadvantage drives B-plot each session. Point-buy
evolution: buying off disadvantages = arc beat. Skill-based
showcases for unique investments.

Per-session touchpoints: disadvantage activation forcing choice,
encounter requiring specific skill/advantage, consequence from
prior disadvantage decision, opportunity to earn points or buy
off disadvantage.
