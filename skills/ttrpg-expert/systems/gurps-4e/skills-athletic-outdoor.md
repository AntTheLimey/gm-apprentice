## Cross-Category Notes

### Riding Specializations

Riding (DX/A, B217) is taken by mount type. Common
specializations: Horse, Camel, Elephant, Dragon, etc.
Each defaults to the others at -3. Riding (Horse) defaults
to Animal Handling (Equines)-3. Combat riding uses the Mount
skill (DX/A, B210), which is distinct from Riding.

### Navigation Specializations

Navigation (IQ/A, B211) specializations default to each other
at -2 (similar media) or -5 (dissimilar). Land and Sea are the
most common. All are TL-dependent except Land at low TL. Also
listed under Technical and Vehicle.

### Swimming and Scuba

Swimming (HT/E) covers surface swimming and basic underwater.
Scuba (IQ/A) is required for compressed-air diving and is
TL-dependent. Both are useful for aquatic Survival situations.

### Animal Handling Defaults

Animal Handling specializations default to each other at -2.
Veterinary defaults to Animal Handling (any)-6. Falconry
defaults to Animal Handling (Raptors)-3.

### Athletic Skills and Combat

Several athletic skills support combat roles:
- **Acrobatics**: +2 to Dodge if successful (one attempt per turn)
- **Climbing**: Access to elevated positions, escape routes
- **Jumping**: Exceed normal jumping limits
- **Throwing**: General throwing ability (also in Ranged Combat)
- **Body Sense**: Reorient after being thrown, teleporting, etc.
