# Pathfinder 2e (Remaster) — Class Feats

Signature class feats for the sixteen Player Core / Player Core 2 classes,
favoring levels 1-8 — the picks that define each class's iconic play patterns.
Action cost is noted in the effect line (one action / two actions / three
actions / reaction / free action). "—" in Prereq means no listed prerequisite
beyond the class itself.

**Coverage:** ~102 of ~904 Player Core-family class feats curated here; the
full set lives in the dataset. Feat names, levels, prerequisites, action
costs, and traits are reproduced as-is; effects are summarized in my own words.
