# RPG Terminology

Terminology reference for tabletop RPGs. Use this file to
resolve system-specific terms and map them to standard
internal terminology.

**System-specific rules:** for mechanics beyond terminology,
read the relevant system reference:
- CoC 7e: `systems/coc-7e/skills.md`, `systems/coc-7e/combat-reference.md`
- D&D 5e: `systems/dnd-5e-2024/conditions-rules.md`
- GURPS 4e: `systems/gurps-4e/combat.md`, `mechanics.md`
- FitD: `systems/fitd/rules-reference.md`

## GM Titles by System

| Title | Abbreviation | Systems |
|-------|:---:|---------|
| Game Master | GM | GURPS, generic |
| Dungeon Master | DM | D&D, d20 System |
| Keeper of Arcane Lore | Keeper | Call of Cthulhu |
| Storyteller | ST | World of Darkness |
| Master of Ceremonies | MC | PbtA games |
| Referee | Ref | OSR, older editions |
| Judge | Judge | DCC |
| Warden | Warden | Mausritter, Cairn |

Use "GM" as the standard term. Recognize all titles as the
same fundamental role.

## Adventure Terminology

| Term | Meaning |
|------|---------|
| Campaign | Connected series of sessions, months to years |
| Arc / Season | Related adventures, 3-12 sessions |
| Adventure / Chapter | Complete storyline, 1-4 sessions |
| Session | Single play meeting, 3-5 hours |
| Scene / Encounter | Discrete gameplay moment, 15-60 min |
| One-shot | Complete game in a single session |
| Module | Published adventure with maps and stat blocks |
| Scenario | CoC term for a single investigation |

## Character Terms

| Term | Meaning |
|------|---------|
| Backstory | Character history before play begins |
| Stats / Attributes | Numerical innate capabilities (STR, INT, DEX, etc.) |
| Skills | Trained capabilities for specific actions |
| Traits / Feats | Special abilities; GURPS: advantages/disadvantages |
| Level | Overall power in class-based systems |
| XP | Experience points tracking progress |
| Class / Archetype | Role definition (fighter, wizard, playbook) |
| Species / Ancestry | Biological origin (modern term replacing "race") |

## Gameplay Terms

| Term | Meaning |
|------|---------|
| Roll / Check / Test | Dice roll to determine uncertain outcome |
| DC / TN | Difficulty Class / Target Number to meet or exceed |
| Modifier / Bonus | Number added to or subtracted from a roll |
| Critical Success | Exceptional positive result from specific die outcome |
| Critical Failure / Fumble | Exceptional negative result |
| Initiative | Turn order in structured time |
| Round | Complete cycle where all participants act once |
| Turn | One participant's action within a round |
| Encounter | Significant scene requiring mechanical resolution |
| Rest / Downtime | Recovery period; short (minor) or long (major) |
| RAW | Rules As Written (literal text) |
| RAI | Rules As Intended (designer intent) |
| House Rules | Group-specific modifications to official rules |
| Homebrew | Custom content (rules, settings, creatures) |

## Standard Internal Terms

When users employ system-specific terminology, use the standard
term internally while preserving original language in output.

| Standard Term | Description | Instead of |
|--------------|-------------|------------|
| GM | Person running the game | Keeper, DM, Storyteller |
| GM Notes | Private GM information | Keeper Notes, DM Notes |
| Campaign | Connected game spanning sessions | Chronicle, Series |
| Entity | Generic campaign element | Object, Element, Thing |
| Session | Individual play meeting | Game, Meet |
| Timeline Event | Event in campaign chronology | History entry, Log |
