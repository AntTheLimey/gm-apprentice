## Overview

A beacon is an automated signal station placed in hyperspace to assist in navigation through hyperspace. The beacons emit a unique signal that allow ships to lock onto its location. Ships can use the signal to determine direction in hyperspace to the beacon, while using it's signal strength to determine distance. More modern ships navigation systems can use multiple signals to triangulate a rough location in physical space of the ship. Since the power and calculation requirements for gravitational navigation can be immense, many ships operate in hyperspace with beacon only navigation.
