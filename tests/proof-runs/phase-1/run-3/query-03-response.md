# Regency Cthulhu — Available Occupations

Regency Cthulhu uses its own occupation list in place of the standard CoC 7e
list, covering the Regency period (1811–1820). The full list is given below,
organised by social band (Credit Rating range).

**Bold** = suitable for gentry characters.
† = occupation with a unique Regency stat block (see below).
\* = also suitable as a hobby career for a Gentleman or Gentlewoman.

## New Regency-Specific Stat Blocks

Five occupations have stat blocks unique to the Regency setting (the sixth,
Butler/Valet/Maid, uses a base equivalent with period substitutions).

**Gentleman** — Landed gentry; lives by wealth and family privilege.
- Formula: EDU×2 + APP×2 | CR 50–90 (Band 4: Gentry)
- Skills: Charm, Dancing, Etiquette, Fashion, Gaming, Ride, one weapon skill,
  one hobby skill.

**Gentlewoman** — Gentry woman; social standing measured by accomplishments.
- Formula: EDU×2 + APP×2 | CR 50–90 (Band 4: Gentry)
- Skills: Art/Craft (Embroidery/Fine Art/Instrument/Singing), Dancing,
  Etiquette, Fashion, Language (Other), Persuade, Ride, one hobby skill.

**Nouveau Riche** — Wealthy through trade or industry; aspires to gentry status.
- Formula: EDU×4 | CR 50–90 (Band 4: Gentry)
- Skills: Accounting, Appraise, Art/Craft (Business), Dancing, Fast Talk,
  Persuade, Ride, one weapon skill.

**Servant (Footman)** — Male household servant; errands and public-facing duties.
- Formula: INT×2 + POW×2 | CR 10–30 (Band 1: Laborer)
- Skills: Animal Handling, Art/Craft (Any), Dodge, Etiquette, Fast Talk,
  Listen, Ride, Spot Hidden.

**Servant (Housemaid)** — Female domestic servant; cleaning and personal service.
- Formula: INT×2 + POW×2 | CR 10–30 (Band 1: Laborer)
- Skills: Art/Craft (Cooking/Sewing), Dodge, Etiquette, Fashion, Fast Talk,
  Listen, Spot Hidden, Stealth.

**Note:** Butler/Valet/Maid CR is capped at 40% regardless of band (reflects
the elevated status of senior household staff). Band 2 artistic occupations
(Actor, Artist, Entertainer, Musician, etc.) may raise CR up to 70% with
Keeper permission.

## Keeper Notes

- Occupations **not** on the Regency list use the nearest base CoC 7e
  equivalent with period-appropriate skill substitutions (Keeper approval
  required). For example, there is no "Police Officer" in the Regency period —
  substitute a Bow Street Runner using the Police Officer or Military Officer
  base block, swapping modern skills for period equivalents.
- The Gentleman and Gentlewoman occupations allow hobby careers: a Gentleman
  may also practice as an Antiquarian, Archaeologist, Artist, Author, Burglar,
  Gambler, Librarian, Occultist, Researcher, Smuggler, or Spy, using that
  occupation's skills but retaining the Gentry CR band (50–90%).
- Skill point formulas follow the base CoC 7e table (see base `occupations.md`):
  EDU×4, EDU×2+APP×2, EDU×2+DEX×2, etc. All characteristic values are
  percentile (×5), not raw 3–18.
- Personal interest points (INT×2) go to any skill except Cthulhu Mythos,
  as per standard CoC 7e.
