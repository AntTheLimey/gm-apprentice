## Fortune Rolls in Blades in the Dark

A **Fortune Roll** is one of the four core roll types in Blades in the Dark. Its purpose is to take a decision out of the GM's hands — letting the dice determine how something unfolds when no PC is directly driving the action.

### When to Use a Fortune Roll

Fortune rolls come up when:

- There is **no obstacle** for a PC to overcome directly (so an action roll is inappropriate), but the outcome is uncertain enough that the GM doesn't want to just decide it.
- You want to determine the **quality of information** gathered (see Gathering Information below).
- The **Engagement Roll** is made at the start of a score to set the crew's opening position.
- **Downtime activities** (recover, acquire asset, reduce heat, etc.) use the same result ladder.
- An **NPC or faction** acts and the GM wants to disclaim how well it goes.

### How to Build the Dice Pool

Roll a number of d6s equal to the relevant **trait rating** — this is usually an action rating, an attribute rating, a Tier rating, or simply 1d as a base with modifiers added or subtracted.

There is no hard cap: start from the base and add or subtract dice based on circumstances, advantages, or disadvantages.

### Reading the Result (always take the single highest die)

| Result | Outcome |
|:---:|---------|
| **Crit** (two+ 6s) | Exceptional / extreme effect |
| **6** | Good / standard effect |
| **4–5** | Mixed / limited effect |
| **1–3** | Bad / poor effect |

If you have zero dice in the pool, roll 2d6 and take the **lowest** result (this pool cannot crit).

### Key Use Cases

**Gathering Information**
When a PC wants information and there is no obstacle to overcome (and it is not common knowledge), call for a fortune roll. The effect level determines detail:
- Great effect → exceptional detail
- Standard effect → good detail
- Limited effect → partial/incomplete detail

Complex questions work better as a **long-term project clock** during downtime rather than a single roll.

**The Engagement Roll**
At the start of every score, the GM calls for an engagement fortune roll starting at **1d**, modified up or down by plan quality, target vulnerability, crew relationships, and Tier differences. The result sets the crew's opening position:

| Result | Opening Position |
|:---:|-----------------|
| Crit | Controlled; first obstacle already overcome |
| 6 | Controlled |
| 4–5 | Risky |
| 1–3 | Desperate |

**Downtime Rolls**
Downtime activities (Acquire Asset, Long-Term Project, Recover, Reduce Heat) use the same result ladder:
- 1–3: one tick / minimal result
- 4–5: two ticks / partial result
- 6: three ticks / full result
- Crit: five ticks / exceptional result

A PC can spend **1 coin** after the roll to increase the result by one level.

**NPC and Faction Actions**
When an NPC or faction acts off-screen and the GM wants to determine how well it goes without deciding personally, a fortune roll on an appropriate Tier or trait rating lets the fiction answer the question.

### Fortune Roll vs. Action Roll

The core distinction: an **action roll** is made by a PC who is actively doing something dangerous or troublesome — the PC is on the hook for consequences. A **fortune roll** removes that authorship from both GM and player; it is used specifically to **disclaim decision-making** to the dice. There are no position/effect negotiations, no resistance rolls against its consequences, and no Devil's Bargains on a fortune roll.
