# Fortune Rolls in Blades in the Dark

A **fortune roll** is used whenever the GM wants to disclaim a decision to the dice — that is, when an outcome should be determined by chance, fate, or a factor outside any PC's direct action. It is one of the four core roll types in Blades in the Dark (alongside action rolls, resistance rolls, and downtime rolls).

## When to Use a Fortune Roll

Fortune rolls apply in three broad situations:

1. **Gathering information with no obstacle and no common knowledge** — if there's nothing actively blocking the PCs but the answer isn't just known, roll fortune for the quality of the information obtained.
2. **NPC or faction activity** — to determine how well an NPC, cohort, or faction fares in an off-screen endeavor.
3. **Environmental or random outcomes** — anything the GM wants the dice to settle rather than deciding unilaterally (e.g., "how bad is the storm tonight?" or "does the contact show up?").

## Building the Dice Pool

Roll a number of d6s equal to a relevant **trait rating** plus any applicable bonuses. The trait can be:

- A **PC's action rating** (e.g., Consort, Hunt) if a PC's capability is a factor
- A **faction's Tier** if the roll represents organizational strength
- A **cohort's quality** for an allied gang or expert
- **1d** as a neutral baseline (modified up or down by circumstances), which is also how the **engagement roll** starts

## The Result Table

| Result | Outcome |
|:---:|---------|
| Crit (two+ 6s) | Exceptional / extreme effect |
| 6 | Good / standard effect |
| 4-5 | Mixed / limited effect |
| 1-3 | Bad / poor effect |

This is the same progression used for downtime rolls.

## Zero Dice

If penalties reduce the pool to zero, roll 2d6 and take the **lowest** result. A crit is not possible at zero dice.

## The Engagement Roll

The engagement roll at the start of every score is a special fortune roll. It starts at **1d** and is modified by plan factors:

| Factor | Modifier |
|--------|:--------:|
| Bold or daring plan | +1d |
| Overly complex/contingent plan | -1d |
| Detail exposes a vulnerability in the target | +1d |
| Target's strength aligns against this approach | -1d |
| Friends or contacts assist | +1d |
| Enemies or rivals interfere | -1d |
| Other contextual advantage (e.g., lower-Tier target) | +1d |
| Other contextual disadvantage (e.g., higher-Tier target) | -1d |

**Engagement results:**

| Result | Opening Position |
|:---:|-----------------|
| Crit | Controlled; first obstacle already overcome |
| 6 | Controlled |
| 4-5 | Risky (the default) |
| 1-3 | Desperate |

After the roll, cut straight to the first serious obstacle — do not narrate the approach. The engagement result sets the starting position only; all subsequent rolls follow normal position rules.

## Fortune Rolls During Flashbacks

Flashbacks can involve fortune rolls when a past action had an uncertain outcome. A failed flashback produces a present-tense consequence (the action happened, but it went wrong and the complication is now in play).

## Key Design Principle

Fortune rolls exist so the GM does not have to invent outcomes unilaterally. They are a **narration tool**, not a challenge mechanic — there is no position, no effect rating, and no stress cost attached to a fortune roll. The result simply tells you how well the relevant force performed at whatever it was doing.
