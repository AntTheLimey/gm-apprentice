# Regency Cthulhu Occupations

## New Regency Stat Blocks

**Gentleman** — Landed gentry; lives by wealth and privilege.
- Formula: EDU×2 + APP×2 | CR 50–90
- Skills: Charm, Dancing, Etiquette, Fashion, Gaming, Ride, one weapon skill, one hobby.

**Gentlewoman** — Gentry woman; accomplishments define social standing.
- Formula: EDU×2 + APP×2 | CR 50–90
- Skills: Art/Craft (Embroidery/Fine Art/Instrument/Singing), Dancing, Etiquette, Fashion, Language (Other), Persuade, Ride, one hobby.

**Nouveau Riche** — Wealthy by trade or industry; seeks gentry acceptance.
- Formula: EDU×4 | CR 50–90
- Skills: Accounting, Appraise, Art/Craft (Business), Dancing, Fast Talk, Persuade, Ride, one weapon skill.

**Servant (Footman)** — Male household servant; errands and public duties.
- Formula: INT×2 + POW×2 | CR 10–30
- Skills: Animal Handling, Art/Craft (Any), Dodge, Etiquette, Fast Talk, Listen, Ride, Spot Hidden.

**Servant (Housemaid)** — Female domestic servant; cleaning and personal service.
- Formula: INT×2 + POW×2 | CR 10–30
- Skills: Art/Craft (Cooking/Sewing), Dodge, Etiquette, Fashion, Fast Talk, Listen, Spot Hidden, Stealth.

- Butler/Valet/Maid CR is capped at 40% (reflects senior household staff status).
- Band 2 artistic occupations may raise CR to 70% with Keeper permission.
- Gentry-suitable occupations (*) can be used as hobby careers within the Gentleman/Gentlewoman framework, keeping the gentry CR band.

## Using Occupations Not in the Regency List

If a player concept doesn't match any listed occupation, use the nearest base CoC 7e equivalent with period-appropriate skill substitutions agreed with the Keeper. For example:
- Modern "Engineer" → Architect or Craftsperson with appropriate trades
- "Police Detective" has no direct equivalent; consider a Bow Street Runner (use Criminal or Spy framework)
