## Fortune Rolls in Blades in the Dark

A Fortune Roll hands off a GM decision to the dice. It replaces judgment calls about uncertain situations — NPC reactions, off-screen plans, information quality.

### Building the Pool
Roll d6s equal to the relevant trait rating (action rating, faction Tier, quality level). No relevant trait = 1d baseline.

### Reading Results

| Result | Outcome |
|--------|---------|
| Critical (two+ 6s) | Exceptional / extreme effect |
| 6 | Good / standard effect |
| 4-5 | Mixed / limited effect |
| 1-3 | Bad / poor effect |

### When to Use
- Gathering information with no obstacle
- Off-screen NPC activity (roll their Tier)
- Uncertain environmental outcomes
- Downtime rolls (same result scale)

### Key Distinction from Action Rolls
No position, no effect level, no resistance possible. The result simply describes what happens.
