## GURPS 4e Grappling

Multi-step contested process using DX or a dedicated skill (Brawling, Wrestling, Judo, Sumo Wrestling).

### Step 1 — Grab
Use Attack maneuver. Roll grappling skill to hit. On success + failed defense, grapple is established. No damage dealt. Target is at -4 to DX.

### Step 2 — Maintain
Free action each turn. Grappler is also limited — cannot move normally.

### Step 3 — Follow-Up Actions
Quick Contest of ST or grappling skill:

| Action | Effect |
|---|---|
| Takedown | Target falls prone, still grappled |
| Pin | Target pinned, cannot act without breaking free |
| Choke Hold | Deals fatigue damage, can render unconscious |
| Arm Lock | Controls limb, can cripple |
| Throw (Judo) | Target thrown, takes falling damage |

### Step 4 — Break Free
Quick Contest of ST or grappling skill. Success breaks grapple entirely.

### Key Modifiers
- Multiple grapplers add ST scores together
- Grappled = -4 DX to all skill rolls and active defenses
- All-Out Attack gives +4 to grab roll
- Flexibility/Double-Jointed gives bonus to escape
