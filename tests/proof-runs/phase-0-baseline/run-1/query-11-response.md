## Universal Faction Turn

Between-session procedure to advance the world independently of PC action. Runs as Step 2 of the Post-Session Update Checklist.

### The Five Questions (per active faction)

1. **What is their current goal?** (concrete terms)
2. **What would they do if the PCs didn't exist?** (classify impact: Critical/Significant/Minor/Flavour)
3. **Did the PCs affect this faction?** (direct or indirect)
4. **What changes?** (advance, alter, delay, or derail depending on PC interference)
5. **What becomes visible to the PCs?** (directly, through allies, rumour, or off-screen)

### Broader Flow
Sits inside the six-step Post-Session Update Checklist:
1. Thread State Updates → 2. **Faction Turns** → 3. Consequence Surfacing → 4. Foreshadowing Review → 5. Discovery State Updates → 6. World State Changes

### FitD-Specific Additions
- Clock ticking rules based on PC opposition/help
- Tier and Hold changes
- Heat consequences at 4+
- Fortune rolls using Tier as die pool
